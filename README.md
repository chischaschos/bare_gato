# BareGato

This is an engine to play the [**El Gato**](http://en.wikipedia.org/wiki/Tic-tac-toe) game

## Installation

Add this line to your application's Gemfile:

    gem 'bare_gato'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bare_gato
